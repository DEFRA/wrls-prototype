Prepare your bulk returns
-------------------------
Open each file in your chosen spreadsheet application
Your returns are shown in columns.  Fill in every empty cell in the table
Figures must be expressed as abstraction volumes in cubic metres only
Export your returns as CSV files

Upload your bulk returns
------------------------
Go to https://wrls-prototype.herokuapp.com/ex/upload-bulk to submit your bulk returns
Make sure you complete and submit each CSV file contained in this folder
